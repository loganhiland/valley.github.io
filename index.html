Workflow Notes

  1. Run the MOOSE parameter sweep.
  2. Postprocess results in matplotlib.
  3. Export matplotlib plots as PNG or WebP files.
  4. Export matching 2D temperature-distribution images.
  5. Place images in the correct figures/ folders.
  6. Update case definitions in script.js.
